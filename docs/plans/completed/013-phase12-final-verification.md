## 발견한 문서 불일치 요약 (각 Phase 진행 기록에 상세 기록, 여기서는 목록만)

- PRD.md §8 `Order` dataclass 예시의 필드 순서가 Python 문법상 오류(002-phase1). 코드에서 `created_at`을 `status`보다 앞에 배치해 수정, PRD.md 본문은 사소한 오타로 판단해 변경하지 않음.
- SPEC.md §4.2/§6.1 예시 화면 수치가 §8 공식으로 재계산한 값과 다름(004-phase3). SPEC.md §8 "계산식이 Single Source of Truth" 원칙에 따라 공식을 그대로 구현, 예시 수치(문서)는 변경하지 않음.
- ARCHITECTURE.md §4.2 `advance()` 의사코드의 `queue[0].started_at = now`를 `= item.started_at + total_time`으로 수정(008-phase7) — 문서도 함께 갱신 완료.
- 생산라인 큐가 메모리 전용이라 재시작 시 유실되는 설계 공백 발견(011-phase10) — 000-overview.md §11에 남은 위험으로 기록, 이번 계획 범위에서는 수정하지 않음.
